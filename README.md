# FlashMaster Frontend

A React.js frontend for the FlashMaster Exam Helper Application.

## Tech Stack
- React.js
- React Router DOM
- Axios
- Vercel

## Live URL
https://flashmaster-frontend-qmf9.vercel.app

## Admin Portal
https://flashmaster-frontend-qmf9.vercel.app/admin-login

## Pages
- /login — Student login
- /register — Register new account
- /dashboard — Main hub
- /materials — Upload and view files
- /flashcards — Create flashcards
- /plan — Study planner
- /progress — Track progress
- /admin-login — Admin portal
- /admin — Admin dashboard

## Features
- JWT authentication
- File upload to Cloudinary
- Flashcard system
- Study planner
- Progress tracker
- Role based access
- Live on Vercel

## Installation
npm install
npm start
