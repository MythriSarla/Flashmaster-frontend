# FlashMaster Frontend

A React.js frontend for the FlashMaster Exam Helper Application — a smart digital study assistant for students.

## Tech Stack
- React.js — Frontend framework
- React Router DOM — Page navigation
- Axios — API calls to backend
- Vercel — Deployment platform

## Live App URL
https://flashmaster-frontend-qmf9.vercel.app

## Admin Portal
https://flashmaster-frontend-qmf9.vercel.app/admin-login

## Pages
| Page | Route | Description |
|------|-------|-------------|
| Login | /login | Student login page |
| Register | /register | New user registration |
| Dashboard | /dashboard | Main navigation hub |
| Materials | /materials | Upload and view study files |
| Flashcards | /flashcards | Create and revise flashcards |
| Study Plan | /plan | Set exam date and study hours |
| Progress | /progress | Track revision progress |
| Admin Login | /admin-login | Separate admin portal |
| Admin Dashboard | /admin | Monitor users and content |

## Features
- JWT based secure authentication
- Upload PDF, PPTX, DOCX, images to Cloudinary
- Flip flashcards with difficulty tracking
- Personalized study planner
- Progress tracker with completion percentage
- Role based access — Student and Admin
- Fully responsive UI
- Live deployment on Vercel

## Installation
```bash
npm install
npm start
```
